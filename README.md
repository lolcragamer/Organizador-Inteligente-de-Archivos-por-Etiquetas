## 🎯 ¿Qué hace esta herramienta?

**Imagina que tienes 100 archivos desordenados y quieres:** 
- `factura_enero.pdf` → `contabilidad/2024/enero/`
- `foto_vacaciones.jpg` → `fotos/2024/vacaciones/`
- `informe_cliente.docx` → `trabajo/proyectos/clienteX/`

**Con esta app, solo escribes la ruta como etiqueta y los archivos se organizan solos.**
